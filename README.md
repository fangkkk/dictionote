# DictiNote - 简单的便签软件

一个使用 PyQt6 开发的便签软件。

## 许可证

本项目使用 [GNU General Public License v3.0](LICENSE) 开源许可证。

由于使用了 PyQt6，根据 GPL 许可要求，本项目必须使用 GPL 许可证开源。

## 功能特点

- 按日期管理便签
- 支持未来日期规划
- 自定义字体和颜色
- 自动保存
- 简洁的界面

## 安装方法

### 方法一：直接下载可执行文件

1. 从 [Releases](https://github.com/yourusername/dictionote/releases) 页面下载最新版本
2. 解压后直接运行 `DictiNote.exe`
